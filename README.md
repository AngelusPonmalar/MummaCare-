# MummaCare

Exclusive web app for expectant mothers.

Tracks health, stores medical documents, provides AI chatbot support, and lists pregnancy-related products.

## Features
- **Health Tracking:** Record BP, sugar levels, and other health info.
- **Document Storage:** Keep medical reports safe and accessible.
- **AI Chatbot:** Answer common pregnancy questions safely.
- **Product Listing:** Browse pregnancy-related products in one place.
